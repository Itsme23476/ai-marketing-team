# Marketing Report (Markdown)

The compiler for `/market report <url>`. You gather every prior analysis on file, score the
site across the six standard dimensions, and produce one client-ready `MARKETING-REPORT.md`
with a prioritized action plan. This is the `cmo-strategist` deliverable.

## Before you start

Load `~/.claude/skills/marketing-team/references/playbook.md`, the brand brief, and the
scoring methodology — **do not duplicate the scoring math here**, load it:
`~/.claude/skills/marketing-team/references/scoring.md` (six weighted dimensions, grade bands,
recommendation tiers, honest-ROI impact labeling). Work as `cmo-strategist`. Label data tiers
(Rule 2) and never fabricate revenue (Rule 3).

## Step 1 — Collect existing data (fold in the cwd)

Check the current directory and fold in whatever exists rather than re-analyzing:
- `MARKETING-AUDIT.md` (primary — scores + findings)
- `COMPETITOR-REPORT.md` (competitor comparison)
- `BRAND-VOICE.md` (voice/messaging assessment)
- `FUNNEL-ANALYSIS.md`, `SEO-AUDIT.md`, `LANDING-CRO.md`, and any `*-AUDIT.md` present

If none exist, fetch the URL (Tier-1 tools if connected, else `WebFetch` +
`python3 ~/.claude/skills/marketing-team/scripts/analyze_page.py <url>`), or offer to run
`/market audit` first for a deeper result. Be transparent about what is measured vs. estimated.

## Step 2 — Score

Apply the six weighted dimensions and grade bands from `scoring.md`
(Content 25% · Conversion 20% · SEO 20% · Competitive 15% · Brand 10% · Growth 10%). Reuse
scores from `MARKETING-AUDIT.md` if present; only re-derive what's missing. Show the weighted
composite and the letter grade.

## Step 3 — Category deep-dives

For each of the six dimensions: score + grade, 3–5 evidence-backed findings (quote real page
copy), what's working, gaps, prioritized recommendations, and impact labeled per `scoring.md`
(🔴/🟡/🟢 leverage; dollars only from real inputs with the formula shown, otherwise a benchmark
range marked `(illustrative)`).

## Step 4 — Supporting summaries

- **Competitor comparison** — fold in `COMPETITOR-REPORT.md` as a positioning matrix if present.
- **Brand & voice** — fold in `BRAND-VOICE.md` consistency notes if present.
- **Conversion summary** — fold in `FUNNEL-ANALYSIS.md`/`LANDING-CRO.md` leaks and quick wins.
- **SEO snapshot** — scannable health checklist (titles, meta, headers, schema, speed, mobile).

## Step 5 — Action plan & roadmap

Classify all recommendations into Quick Wins / Strategic / Long-Term per `scoring.md`, each
with impact, effort, and expected result. Then lay out a 30-60-90 day roadmap.
