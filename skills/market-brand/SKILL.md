# Brand Voice Analysis & Guidelines

The voice engine for `/market brand <url>`. You read how the brand actually communicates,
score it across four dimensions, and produce guidelines any writer can follow to stay on-brand.

## Before you start

Load `~/.claude/skills/marketing-team/references/playbook.md` and the brand brief, then dispatch
the **`copywriter`** subagent to do the analysis — scoped to the brand brief. The brief states
the *intended* voice and audience; your job is to document the *actual* voice and flag any gap
between them. Apply the playbook rules; don't restate them.

## Phase 1 — Gather source material

Analyze in priority order. Use a Tier-1 MCP tool if connected (playbook Rule 2), else `WebFetch`
/ `WebSearch`; for structured facts run
`python3 ~/.claude/skills/marketing-team/scripts/analyze_page.py <url>` and label the tier.

- **Primary (required):** homepage, about page, product/service pages.
- **Secondary (if available):** 3–5 recent blog posts, social bios/posts, newsletters, microcopy
  (errors, onboarding, help docs).
- **Tertiary:** job postings, press releases, ad copy, video/podcast transcripts.

**Every assessment needs evidence** — quote 3–5 real examples per rating.

## Phase 2 — Score the four dimensions (1–10)

Rate each as a spectrum, with quoted evidence:

1. **Formal ↔ Casual** — contractions, sentence length, pronouns, greetings, slang.
2. **Serious ↔ Playful** — humor, exclamation/emoji use, wordplay, error-message tone.
3. **Technical ↔ Simple** — jargon, acronyms, detail level, audience assumed.
4. **Reserved ↔ Bold** — hedged vs direct claims, opinions, competitive references, ambition.

Then map **tone by context** (homepage, product, blog, social, 404, subject lines, CTAs,
support) with a quote each — voice is constant, tone shifts.

## Phase 3 — Personality, vocabulary, consistency

- **Archetype** (blend 1–2): Authority · Innovator · Friend · Rebel · Guide. State primary,
  secondary, and fit (strong/moderate/weak) with evidence.
- **Vocabulary** — 15–20 characteristic words grouped into action / descriptive / value /
  industry terms; the words they avoid; any signature phrases or taglines.
- **Consistency audit** — rate each channel consistent/mostly/inconsistent with notes; give an
  overall score X/10. Frame gaps as fixable opportunities, not failures.
- **Competitor comparison** (if `COMPETITOR-REPORT.md` exists or 2–3 rivals are easy to fetch) —
  plot all four dimensions in a matrix and name the unoccupied voice territory.

## Phase 4 — Build the usable guide

- **Messaging hierarchy** — tagline (<10 words) → 3–5 value props → elevator pitch (~75 words)
  → boilerplate (100–150 words) → note whether a full brand story exists.
- **Voice chart** — "Our voice IS / IS NOT" (e.g. Confident not Arrogant; Clear not Dumbed-down).
- **Do's & Don'ts** — specific, derived from the analysis, not generic.
- **Copy samples (8)** — the most valuable section; people learn voice by example. Write a
  headline, product paragraph, blog opening, social post, subject line, CTA, error message, and
  thank-you message in the identified voice, consistent with the brand brief.
