# Marketing Audit (Flagship)

The full-team review for `/market audit <url>`. You dispatch the specialists in parallel,
aggregate their findings, score the site across six dimensions, and produce a client-ready
`MARKETING-AUDIT.md`.

## Before you start

Load `~/.claude/skills/marketing-team/references/playbook.md` and the brand brief. Also load
`~/.claude/skills/marketing-team/references/scoring.md` for the scoring method. Apply honest
ROI (no invented dollars) and label data tiers throughout.

## Phase 1 — Discovery

1. **Fetch the site.** Use `WebFetch` on the homepage plus up to 5 key interior pages
   (pricing, about, product/features, blog, contact).
2. **Run structured extraction** for hard facts (SEO tags, CTAs, forms, tracking pixels,
   schema):
   ```bash
   python3 ~/.claude/skills/marketing-team/scripts/analyze_page.py <url>
   ```
   If unavailable, fall back to WebFetch parsing and note reduced precision.
3. **Detect business type** (playbook Rule 5) — this shapes every specialist's focus.
4. **Build a page map** (home, pricing, product, about, blog, signup/contact, legal) and pass
   it to every specialist.

## Phase 2 — Dispatch the team (parallel)

Launch these subagents simultaneously, each given the brand brief, business type, page map,
and fetched content. Tell each to use Tier-1 data tools if connected (playbook Rule 2).

| Subagent | Scores | Focus |
|----------|--------|-------|
| `copywriter` | Content & Messaging | Headline clarity, value prop, persuasion, CTAs, voice |
| `performance-marketer` | Conversion Optimization | CTAs, forms, friction, social proof, pricing page |
| `seo-lead` | SEO & Discoverability | On-page + technical SEO, structure, keyword coverage |
| `competitive-analyst` | Competitive Positioning | Differentiation, comparison pages, market category |
| `cmo-strategist` | Brand & Trust + Growth & Strategy | Trust signals, pricing, growth loops, retention |

Each returns a 0–100 score for its dimension(s) plus wins, fixes, and before/after examples.

## Phase 3 — Synthesize

1. **Compute the composite score** using the weights in `scoring.md`.
2. **Hand all findings to `cmo-strategist`** to produce the strategic summary and the single
   prioritized action list (🔴/🟡/🟢) — don't just concatenate the specialists.
3. **Classify recommendations** into Quick Wins / Strategic / Long-Term (`scoring.md`).
4. **Impact** — label each by leverage. Only attach numbers if real inputs exist; otherwise
   use benchmark ranges marked `(illustrative)`.
5. **Competitor comparison table** if the analyst found rivals.

## Error handling

- URL unreachable → report it, suggest checking the URL.
- A specialist fails → continue with the rest, note the gap in the report.
- Auth-gated site → analyze what's accessible, flag the rest for manual review.
- Thin/single-page site → adapt scope and say so.

## Chaining

Fold in `COMPETITOR-REPORT.md` or `BRAND-VOICE.md` if present. Suggest `/market proposal`
(to sell the fixes) or `/market copy` / `/market funnel` (to start implementing) as next steps.
