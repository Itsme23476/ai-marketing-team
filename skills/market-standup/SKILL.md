# Marketing Team Standup (Flagship)

The "whole team in one command" deliverable for `/market standup`. You simulate a real team
standup: every specialist contributes their single highest-leverage move, then the CMO
synthesizes everything into one prioritized 90-day plan.

## Before you start

Load `~/.claude/skills/marketing-team/references/playbook.md` and the brand brief (playbook
Rule 1). If no brand brief exists, offer to create one first — a standup is only as good as the
business context behind it. Label data tiers (Rule 2) and never fabricate numbers (Rule 3).

## Step 1 — Establish context

1. Determine the target: a URL from the `/market standup` argument, or the website in the brand
   brief. If neither exists, ask for one (or run a brief-only strategic standup).
2. If a URL is available, fetch it and run the analyzer for hard facts:
   ```bash
   python3 ~/.claude/skills/marketing-team/scripts/analyze_page.py <url>
   ```
3. Detect the business type (playbook Rule 5).

## Step 2 — Run the team in parallel

Dispatch all seven specialists at once, each scoped to the brand brief, business type, and
fetched data. Tell each to use Tier-1 MCP data if connected, else Tier-0, and to return only
its **top moves**, not an exhaustive report:

| Specialist | Contributes |
|-----------|-------------|
| `cmo-strategist` | Top priorities, positioning, the 90-day spine |
| `seo-lead` | Top organic opportunities (keyword data if Tier-1 available) |
| `copywriter` | The 3 highest-leverage copy rewrites |
| `performance-marketer` | Funnel gaps + one paid-channel test |
| `social-manager` | A 2-week content angle plan |
| `email-marketer` | The one email sequence to build first |
| `competitive-analyst` | Where the brand wins/loses vs 2–3 rivals |

## Step 3 — Synthesize (CMO)

Hand all seven contributions to `cmo-strategist` to produce ONE plan — don't just concatenate.
The synthesis must tie every action to the brand brief's 90-day goal and primary metric, and
sequence them (what unlocks what).

## Notes

- A specialist that fails → continue with the rest, note the gap.
- Keep each specialist's section short; the value is the synthesized plan, not seven reports.
- End by pointing the user to the specific `/market` command that executes move #1.
